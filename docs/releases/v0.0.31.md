# Heddle v0.0.31

Release date: 2026-04-26

## Summary

Heddle v0.0.31 improves confidence around the browser control plane by adding a first Playwright e2e harness and making the session review surface easier to inspect.

This release is focused on release confidence and operator review: the control plane now has automated coverage for core routes and workspace flows, and the session review panel can show richer current-workspace and captured-turn diffs without crowding the main session screen.

## Highlights

- Added a Playwright e2e harness for the local control plane
- Covered overview, route persistence, workspace switching, current git diff visibility, and mobile navigation in e2e tests
- Extracted the session review panel into its own control-plane module
- Added full-diff review behavior for current workspace changes and captured turn diffs
- Updated the README review screenshot to show the expanded diff review flow
- Adjusted TypeScript and ESLint config so the Playwright harness is part of normal development without leaking into the runtime build

## What's New

### Control-plane e2e harness

- Added `yarn e2e`, `yarn e2e:ui`, `yarn e2e:headed`, and `yarn e2e:install`
- Added a fixture daemon script that prepares isolated Git workspaces under `.e2e/`
- Added Playwright coverage for important control-plane navigation and workspace review flows
- Documented the e2e workflow in the development guide

### Review panel usability

- Split review-panel logic out of `SessionsScreen` so the main session screen is smaller and easier to maintain
- Fetches current workspace file diffs for the full changed-file set so the expanded diff dialog can show the whole current workspace patch
- Keeps turn-history review, current workspace review, and evidence review separated behind explicit tabs
- Simplifies diff line-number display for compact review panes

## Upgrade Notes

- Package version is now `0.0.31`
- No breaking CLI flag or config migration is introduced in this release
- Playwright browsers are not installed automatically; use `yarn e2e:install` before running the new e2e suite locally if Chromium is missing

## Known Limits

- The new e2e harness is local-dev coverage, not yet a required CI gate
- The review panel improves current/captured diff inspection, but it remains a review surface rather than a full editor or repository browser
