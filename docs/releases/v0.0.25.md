# Heddle v0.0.25

Release date: 2026-04-19

## Summary

Heddle v0.0.25 introduces a mobile-first control plane experience, splits the heartbeat and chat subsystems into cleaner modules, and adds layout snapshot tooling for UI development.

## Highlights

- Mobile control plane with dedicated shell, session navigation, and chat composer
- Heartbeat CLI and scheduler split into focused sub-modules
- Chat formatting and hooks refactored for maintainability
- Layout snapshot service and debug tooling for control plane UI development

## What's New

### Mobile control plane

- New mobile shell with bottom navigation and responsive layout
- Mobile session navigation with swipe-friendly task and session lists
- Mobile chat composer with prompt drafting and submission
- Mobile-aware design system primitives (badge, button, textarea)
- `useIsMobile` hook for responsive behavior throughout the control plane

### Heartbeat modularization

- Heartbeat CLI split into args, duration, output, run-commands, store, summary, task-commands, types, and worker modules
- Heartbeat scheduler runtime extracted into its own module with task state and task store
- Lucid-specific heartbeat integration kept in a thin adapter

### Chat refactoring

- Chat app hooks refactored: `useChatDrift`, `useChatPickers`, `useLocalIds`, `usePromptDraft`, `usePromptSubmission`
- Core chat formatting split into `conversation-lines`, `failure-messages`, and `trace-summary` modules
- Chat drift detection moved to its own hook

### Developer tooling

- Layout snapshot service for capturing and comparing control plane UI states
- Layout snapshot debug page with visual diff tooling
- Control plane layout screenshots added to docs

## Fixes

- Package scripts organized and renamed for clarity
- CLI entry point used consistently for chat dev scripts
- Roadmap script names updated to match current conventions

## Upgrade Notes

- No breaking CLI changes in this release
- The heartbeat CLI sub-commands remain the same; only internal module layout changed
- If you import from internal chat or heartbeat paths directly, update your imports to the new module locations

## Known Limits

- The mobile control plane is an early first pass; layout and navigation will continue to evolve
- Layout snapshot tooling is intended for development use only
