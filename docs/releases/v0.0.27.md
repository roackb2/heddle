# Heddle v0.0.27

Release date: 2026-04-21

## Summary

Heddle v0.0.27 adds archive-backed chat compaction, makes `ask` usable for multi-turn session testing, and improves operator tooling across both the terminal chat UI and the browser control plane.

## Highlights

- Chat sessions now compact into archived transcripts plus a rolling summary instead of only truncating older history
- `heddle ask` can now create, continue, and reuse saved chat sessions for lightweight testing and automation
- Terminal chat adds `/debug tui-snapshot` so the current TUI view can be saved locally for debugging
- Control plane heartbeat tasks can now be enabled, disabled, and manually triggered from the browser UI

## What's New

### Archive-backed compaction

- Chat compaction now writes archived transcript chunks under `.heddle/chat-sessions/<session-id>/archives/`
- Each compaction pass maintains a rolling summary and archive manifest so future compactions build on prior archived history
- Preflight compaction now runs before oversized session submissions so long-running sessions are less likely to fail at the provider context limit
- Compaction status is surfaced across CLI chat, desktop control plane, and mobile control plane

### Session-aware ask mode

- `heddle ask` now supports:
  - `--session <id>`
  - `--latest`
  - `--new-session [name]`
- Session-backed ask runs reuse the same persistence and compaction flow as interactive chat
- This gives operators a practical non-TUI path for testing session behavior and long-context handling

### TUI debug snapshots

- Terminal chat now supports `/debug tui-snapshot`
- Each snapshot writes:
  - a clean text view of the current chat state
  - a raw ANSI capture
  - metadata including session, model, and terminal size
- This is intended for debugging chat layout, status indicators, approvals, and compaction behavior without screen recording

### Control plane heartbeat controls

- The browser control plane can now:
  - enable heartbeat tasks
  - disable heartbeat tasks
  - manually trigger a task run
- Heartbeat task views and mobile task screens were updated to surface those controls more cleanly

## Fixes

- Chat compaction no longer relies only on post-run truncation
- Lint now rejects oversized non-test source files over 1000 lines
- Minor internal cleanup removed unused chat/runtime helpers and tightened related tests

## Upgrade Notes

- No breaking CLI commands in this release
- Archive-backed compaction stores archived raw transcript content in full under `.heddle/chat-sessions/<session-id>/archives/`
- If you have internal tooling that reads chat session files directly, account for new archive metadata and compaction status fields

## Known Limits

- TUI snapshots are designed for debugging text state, not pixel-accurate terminal reproduction
- Archive compaction currently stores full archived tool output; redaction and retention policy controls are not part of this release
