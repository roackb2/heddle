# Heddle v0.0.26

Release date: 2026-04-19

## Summary

Heddle v0.0.26 advances the mobile control plane with new workflow screens, an approval sheet, and UX refinements across the mobile experience.

## Highlights

- Mobile task management and review screens for on-the-go session oversight
- Mobile approval sheet for reviewing and resolving tool-use approvals from a phone
- Shared conversation message component extracted for reuse across desktop and mobile
- Toast system replaced with shadcn-based implementation
- Tighter spacing and layout polish in the mobile composer and tab navigation

## What's New

### Mobile workflow screens

- **MobileTasksScreen**: browse and manage tasks within a session from the mobile control plane
- **MobileReviewScreen**: review session conversation history, commands, and approvals on mobile
- **MobileApprovalSheet**: bottom-sheet UI for inspecting pending approvals, with risk-level badges and one-tap approve/deny

### Shared components

- `ConversationMessage` extracted as a reusable component shared between desktop and mobile conversation views
- `SessionsWorkspace` and `HeartbeatWorkspace` simplified by delegating to the shared component

### Toast and styling cleanup

- Legacy custom toast system removed in favor of shadcn toast primitives (`toast.tsx`, `toaster.tsx`, `use-toast.ts`)
- Removed unused legacy CSS rules from `control-plane.css`
- Mobile composer and bottom tab bar spacing tightened for better touch targets

## Fixes

- Mobile chat screen tab spacing corrected for consistent alignment
- Composer padding adjusted to avoid overlap with the bottom navigation bar

## Upgrade Notes

- No breaking CLI changes
- If you imported the old `toasts.tsx` helper from the control plane, switch to the new `use-toast` hook
