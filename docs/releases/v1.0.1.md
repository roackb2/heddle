# Heddle v1.0.1

Release date: 2026-05-08

Heddle v1.0.1 is a patch release focused on making terminal and browser sessions easier to operate while an agent is actively working.

The main theme is visibility and control: users can see streamed thinking/tool progress again, choose reasoning effort more cleanly, edit longer prompts more comfortably, and read assistant markdown in both the terminal and control plane.

## Highlights

### Terminal activity is visible again

Heddle now restores live in-flight activity while a turn is running. Tool calls and completions appear promptly, and OpenAI reasoning-summary stream events are surfaced as dim `Thinking:` progress text.

Why this matters:

- long-running tool-using turns no longer look silent or stuck
- internal thinking remains visible enough to explain what the agent is doing
- thinking text is visually distinct from final assistant answers

### Better prompt editing in terminal chat

The terminal composer now has a wider separator-line layout, multiline input fixes, undo/redo, submitted prompt history navigation, and focused hooks for prompt editing state.

Why this matters:

- long prompts wrap correctly across the available terminal width
- users can recover from prompt-edit mistakes before sending
- repeated or iterative prompts are easier to reuse

### Reasoning effort is a first-class session control

Reasoning effort can now be chosen through a picker-style UX similar to model selection, persists with saved sessions, and is carried through OpenAI request wiring where supported.

Why this matters:

- model and reasoning controls feel consistent
- session switches restore the session's configured reasoning effort
- unsupported or reserved effort values are handled explicitly instead of silently falling back

### Markdown rendering during active sessions

Assistant markdown is rendered more consistently in both the browser control plane and terminal stream. The control plane now has a shared assistant-markdown renderer, and the terminal can render streamed assistant markdown in real time.

Why this matters:

- headings, lists, tasks, links, and code fences are easier to scan
- streamed responses feel closer to final responses
- thinking summaries avoid raw markdown emphasis artifacts

## Other Changes

- Added the official Heddle website, [heddleagent.com](https://heddleagent.com), to the top-level README and npm package metadata.
- Added the MIT license file to the package.
- Updated feature-planning workflow guidance for future coding-agent work.
- Added regression coverage for prompt input behavior, reasoning effort commands, OpenAI OAuth reasoning requests, activity projection, markdown rendering, and conversation-panel streaming.
