# Heddle v3.1.3

Release date: 2026-06-06

Heddle v3.1.3 is a focused reliability and usability release for the shared
control-plane experience. It improves recovery from transient model backend
disconnects, keeps terminal command output from crowding active runs, and adds a
Traditional Chinese README for users who prefer that entry point.

## Highlights

### Model turns retry transient backend failures

Agent model calls now have a bounded retry policy for transient response
failures.

Why this matters:

- temporary backend disconnects are less likely to immediately fail the run
- retry behavior is owned by core model-turn execution instead of individual UI
  clients
- retry status is emitted through the existing event stream so terminal and
  browser surfaces can stay informed

### TUI command output collapses when a run starts

Command result panels in the terminal UI now get out of the way once a new run
begins.

Why this matters:

- slash-command results no longer occupy the main reading area while the agent is
  thinking or using tools
- users can focus on the current run state instead of stale command output
- command display behavior stays in the TUI presentation layer

### Traditional Chinese README

The repository now includes `README.zh-TW.md` and links it from the English
README.

Why this matters:

- Traditional Chinese readers have a current project overview without relying on
  machine translation
- the translated README reflects the current v3 control-plane direction
- the English README remains the default package entry point

## Upgrade Notes

- No command migration is required from v3.1.2.
- `heddle` and `heddle chat` continue to open the API-backed TUI.
- Browser and terminal clients continue to use the shared control-plane path.
