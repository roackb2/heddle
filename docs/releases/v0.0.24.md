# Heddle v0.0.24

Release date: 2026-04-18

## Summary

Heddle v0.0.24 is a small patch release that makes chat runs finish more cleanly by preventing duplicate final assistant messages for stopped runs and locking that behavior in with regression coverage.

## Highlights

- Interrupted or otherwise stopped chat runs now persist exactly one final assistant message
- Regression tests now cover final message persistence for both successful and stopped runs

## Fixes

- Fixed duplicate final chat messages for non-`done` outcomes in the chat runtime
- Preserved the newer live-only handling for intermediate `assistant.turn` updates while keeping only one persisted final summary

## Verification

- `pnpm vitest run src/__tests__/chat-runtime.test.ts`

## Upgrade Notes

- There is no known migration step for this patch release
- This patch is focused on chat transcript correctness and operator-facing run history cleanliness
