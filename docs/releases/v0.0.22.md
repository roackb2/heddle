# Heddle v0.0.22

Release date: 2026-04-17

## Summary

Heddle v0.0.22 improves day-to-day local coding workflows with safer external file access, a more reliable browser control plane, and a much stronger documentation surface for setup, discovery, and ongoing use.

## Highlights

- External file reads outside the active workspace now require explicit approval
- The browser control plane is more stable to run and easier to serve reliably
- The README, docs hub, guides, and reference docs were substantially refreshed for discoverability

## What's New

### Safety and operator control

- External file access now requires explicit approval instead of being treated like a normal in-workspace read
- Mutation follow-up reminder churn was reduced so runs are less noisy after edits

### Browser control plane

- Desktop control plane layout issues were fixed
- Tailscale access to the Vite dev server was enabled for remote local-UI workflows
- Daemon web assets are resolved more reliably so the built control plane is less fragile to serve

### Documentation and discoverability

- The root README was refactored to better explain what Heddle is, who it is for, and how to try it quickly
- The docs hub now gives clearer reading paths for users, contributors, and release preparation
- Feature guides and reference docs were expanded across chat, control plane, heartbeat, knowledge persistence, semantic drift, programmatic use, CLI, config, capabilities, and provider setup

## Fixes

- Fixed daemon web asset resolution for the control plane
- Fixed desktop control plane layout regressions
- Fixed noisy mutation follow-up reminder behavior
- Fixed Vite dev server access for Tailscale-based local networking

## Upgrade Notes

- There is no known migration step for normal CLI usage in this release
- If you use the browser control plane in development, the daemon/server should still be restarted when server-side routes or assets change
- If you rely on reading files outside the active workspace, expect an approval step now

## Release Scope Note

- This draft is based on the intended release boundary since the last explicit version bump (`7d5e624`, `0.0.17`) because the repo has not started annotated release tags yet
- The browser control plane remains early compared with a full IDE-like environment
