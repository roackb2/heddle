# Heddle v3.2.2

Release date: 2026-06-08

Heddle v3.2.2 is a focused terminal usability patch. It fixes a prompt input
regression where ordinary letters such as `a`, `d`, and `c` could be consumed
as empty-prompt shortcuts instead of being inserted into the composer.

## Highlights

### Prompt typing wins over local shortcuts

The terminal composer no longer treats bare printable letters as global
shortcuts while the prompt is active.

Why this matters:

- prompts can start with normal words like "add", "debug", or "check" without
  opening a panel
- users no longer need to understand hidden empty-prompt shortcut state before
  typing
- approval prompts still keep their explicit `[a] approve` and `[d] deny`
  behavior because prompt input is blocked while approval mode owns the keyboard

### Terminal-only actions use slash-style aliases

Activity expansion, diff review, and command-output expansion now use
TUI-local slash-style commands:

- `/a` or `/activity` toggles terminal activity details
- `/d` or `/diff` opens terminal diff review
- `/c` or `/commands` toggles terminal command output

Why this matters:

- the commands are discoverable in the existing slash command hint surface
- short aliases preserve quick terminal operation without stealing typed text
- unavailable local actions show a local command message instead of disappearing
  from hints or falling through to the server slash-command registry

### The local-command boundary is documented

The release adds explicit guidance for contributors about which commands belong
in `cli-v2` and which belong in core/control-plane.

Why this matters:

- TUI-only commands stay limited to Ink-local presentation state
- shared session, runtime, model, auth, heartbeat, compaction, and workspace
  commands remain in core/control-plane slash modules
- future command work has a clearer decision rule instead of growing parallel
  command paths by accident

## Upgrade Notes

- No migration is required from v3.2.1.
- `heddle` and `heddle chat` continue to open the API-backed TUI.
- Existing core slash commands such as `/model`, `/session`, `/drift`, and
  `/compact` continue to use the shared control-plane slash-command path.
